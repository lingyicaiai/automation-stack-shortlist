# Automation Stack Shortlist

一个 90 分钟可落地的微型 Web 工具：输入角色、预算、技术水平和待自动化流程，输出 **3 套工具组合** 和 **7 天落地计划**。

## 为什么做
- Reddit /r/automation 与 ProductHunt 自动化分类都存在大量“该用哪套工具起步”的需求信号。
- 该需求明确、无需登录态、无需付费 API，可快速交付 MVP。

## 本地运行
直接打开 `index.html`。

## 部署
- 平台：Vercel（静态站）
- 配置：`vercel.json`

## 输出内容
1. 3 套工具栈推荐（按匹配度排序）
2. 每套成本与优劣势
3. 7 天落地行动清单
